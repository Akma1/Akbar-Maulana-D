Tugas :
Buatlah webapp dengan menggunakan vue untuk MENCATAT TUGAS KULIAH
Spesifikasi :
1) bisa menambah tugas kuliah
2) bisa menghapus tugas kuliah
3) bisa mengedit tugas kuliah
4) bisa menyimpan di local storage (ketika browser dimatikan data masih tersimpan)[BONUS]

Installation :
-install vue js : https://vuejs.org/v2/guide/installation.html
-download bootstrap : https://getbootstrap.com/

page :
index.php -> form input
data.php -> form data
